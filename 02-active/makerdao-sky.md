## Token Economics

### SKY Token
- **Conversion**: 1 MKR = 24,000 SKY
- **Purpose**: Governance voting, staking rewards
- **Supply**: ~2.4B SKY (from ~100k MKR)

### USDS Stablecoin  
- **Peg**: $1 USD
- **Backing**: Over-collateralized crypto assets
- **Yield**: Sky Savings Rate (SSR) - variable APY

---

## Protocol Mechanics

### Collateralized Debt Positions (CDPs)
1. Deposit collateral (ETH, WBTC, etc.)
2. Mint USDS against collateral
3. Pay stability fee (interest)
4. Liquidation if collateral ratio drops below threshold

### Sky Savings Rate (SSR)
- Deposit USDS, earn yield
- Variable rate set by governance
- Similar to old DAI Savings Rate (DSR)
